# Flask API - Hoot Back-End - Solution

## About

This repo contains solution code for the [Flask API - Hoot Back-End](https://git.generalassemb.ly/modular-curriculum-all-courses/flask-api-hoot-back-end.git) lecture.

## Getting started

Fork and clone this repository to your local machine.

After moving into the cloned directory, activate a new virtual environment:

```bash
pipenv shell
```

Install the dependencies:

```bash
pyenv sync
```

Run the Flask app:

```bash
python app.py
```

To deactivate the virtual environment when you're done, run:

```bash
exit
```
